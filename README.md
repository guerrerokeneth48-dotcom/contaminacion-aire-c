# Sistema de Gestion y Prediccion de Contaminacion del Aire

Proyecto de la materia Programacion 1 (ISWZ1102).

Programa hecho en lenguaje C que monitorea y predice los niveles de
contaminacion del aire en 5 zonas de una ciudad.

## Funciones del programa

1. Monitoreo de la contaminacion actual y comparacion con los limites.
2. Prediccion del PM2.5 de las proximas 24 horas y alertas.
3. Promedios historicos de los ultimos 30 dias.
4. Recomendaciones para reducir la contaminacion.
5. Exportar un reporte a un archivo de texto.

## Como compilar y ejecutar

```
gcc main.c -o programa
./programa
```
